# Bio Stack

## 项目简介
这是一个面向计算生物学与 AI for Science 的轻量自动化情报项目。  
它每天从 GitHub 抓取 Bio-AI 相关新仓库，调用 DeepSeek 生成中文科研情报日报，并通过邮件推送，同时把日报归档到仓库，便于长期回看趋势。

## 这个项目解决什么问题
- 手动刷 GitHub 成本高，容易漏掉低星但高潜力的新项目。
- 仅看 Star 不够，无法及时发现“近期活跃 + 有论文/模型线索”的仓库。
- 每天结论分散在邮件里，不利于后续回顾和积累。

这个项目把“抓取 -> 分析 -> 推送 -> 归档”串成一个每天自动执行的闭环。

## 核心能力
- 双轨抓取：
  - 高星趋势轨：关注近 30 天新建且高热的相关项目。
  - 最新更新轨：关注低星但近期高活跃项目。
- README 摘要增强：为每个仓库抓取 README 片段，提高 AI 分析质量。
- 中文情报生成：自动输出结构化日报（趋势 + 深挖 + 全量项目速览）。
- 邮件推送：支持 SMTP SSL/TLS，多收件人。
- 历史归档：每次运行写入日期文件，并自动更新索引与当日摘要。
