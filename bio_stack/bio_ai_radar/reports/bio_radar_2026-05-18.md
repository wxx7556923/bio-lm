# Bio-AI 科研情报雷达 - 2026-05-18

## 一、Top 高星区技术风向

本周高星项目（stars ≥ 5）呈现出几个鲜明的技术趋势：

1.  **Rust 重写单细胞组学工具成为热点**：`scvelo-rs` 和 `rustscenic` 两个项目分别用 Rust + PyO3 重写了经典的 scVelo（RNA velocity）和 SCENIC（调控网络分析）工具。这标志着单细胞组学领域正在从纯 Python/R 生态向高性能编译语言迁移，追求“算法忠实 + 百倍加速”的工程化目标。**工具化、高性能化**趋势明显。

2.  **LLM Agent 深度渗透基因组学**：`bioagent`（端到端自主研究）、`variant-triage`（临床变异解读）、`agentic-genomics`（可审计变异解释）等项目表明，LLM Agent 不再是简单的聊天机器人，而是开始作为“研究副驾驶”嵌入到具体的基因组学工作流中，如变异过滤、注释和临床解读。**LLM Agent 化**是该方向的核心特征。

3.  **基因组基础模型聚焦作物育种**：`OneGenome-Rice` 是专门为水稻训练的基因组基础模型（1.25B 参数，MoE 架构），可处理百万碱基级序列。这表明基因组语言模型正在从泛用型（如 DNABERT）向**物种特化、应用导向**（如精准育种）发展，**平台化**潜力巨大。

4.  **科学绘图与编码 Agent 基础设施涌现**：`AgentFigureGallery` 为编码 Agent 提供“即插即用”的科学绘图技能，`agentic-coding-intro` 则是一份面向生物信息学家的 Agent 编码入门指南。这反映了社区正在构建**Agent 生态的基础设施**，让生物学家能更高效地利用 AI 编码工具。

5.  **NGS 工具链 Rust 化**：`chelae`（NGS 读段修剪）、`vcfkit`（VCF 处理）、`gffbase`（GFF/GTF 引擎）、`poa-consensus`（部分序比对）等项目均采用 Rust 编写，追求零依赖、高性能、单二进制部署。这是生物信息学工具链**现代化、工程化**的典型表现。

**总结**：本周高星区的核心叙事是“**用现代工程手段（Rust、LLM Agent、基础模型）重构经典生物信息学工具**”，单细胞组学和基因组学是两大主战场。
