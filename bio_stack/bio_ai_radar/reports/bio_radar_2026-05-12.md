# Bio-AI 科研情报雷达 - 2026-05-12

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现出以下三大技术趋势：

1.  **Agent 化与 LLM 驱动的生物信息学工具爆发**：这是最显著的趋势。多个高星项目（如 `bioagent`、`celltype-agent`、`variant-triage`、`GEO_Screener`、`agentic-genomics`）均采用 LLM Agent 模式，将大语言模型（如 Claude、GPT）与生物信息学工具链结合，实现从文献综述、数据筛选、变异解读到细胞类型注释的自动化。这些项目不再仅仅是“调用 API”，而是构建了具有推理、工具调用和记忆能力的自主 Agent 系统。这标志着生物信息学正从“脚本驱动”向“Agent 驱动”转变。

2.  **高性能 Rust/PyO3 重写浪潮**：为突破 Python 生态的性能瓶颈，多个项目选择用 Rust 重写经典生物信息学工具，并通过 PyO3 提供 Python 接口。代表项目包括 `rustscenic`（重写 SCENIC+ 调控网络分析）、`scvelo-rs`（加速 RNA velocity 计算）、`gffbase`（GFF/GTF 引擎）、`vcfkit`（VCF 工具集）和 `poa-consensus`（部分序比对）。这些项目强调“零 Java 依赖”、“单二进制文件”、“pip install 即用”，目标是提供更快的速度和更简单的部署体验。这反映了社区对计算效率和工程化水平的更高要求。

3.  **工具化与平台化并重，多模态初现**：项目分化明显。一方面，`NanoQMRA`、`scnoisemeter`、`cnvturbo` 等专注于解决特定分析问题（微生物风险评估、单细胞技术噪音量化、CNV 分析），是典型的“工具化”项目。另一方面，`bioagent` 和 `PharmaCore` 则试图构建端到端的平台（全自动研究平台、本地药物发现平台）。此外，`celltype-agent` 已支持单细胞转录组、空间转录组（Visium）和空间原位（Xenium）三种数据模态，体现了向“多模态”发展的趋势。

**趋势判断**：当前 Bio-AI 领域正处于“Agent 化”和“高性能化”的双轮驱动期。LLM Agent 正在重塑生物信息学工作流，而 Rust 重写则解决了底层性能痛点。未来，将 Agent 的智能决策能力与 Rust 的高性能计算能力结合，可能会成为下一代生物信息学基础设施的标准范式。

### 值得继续跟踪的项目清单
