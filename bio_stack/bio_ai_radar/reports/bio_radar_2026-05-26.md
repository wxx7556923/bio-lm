# Bio-AI 科研情报雷达 - 2026-05-26

## 一、Top 高星区技术风向

本期高星项目（≥4 stars）共19个，呈现出以下技术趋势：

**1. 基因组基础模型进入“规模化+专业化”双轨并行阶段**
- Hugging Face 推出的 **Carbon**（159 stars）以1T tokens/6T碱基对训练，代表了基因组基础模型向超大规模预训练语料库发展的趋势，同时提供了零样本评估和微调脚本，推动标准化评估。
- BGI 杭州AI研究院的 **Genos-m**（12 stars）则聚焦于人类微生物组这一垂直领域，表明基因组基础模型正从“通用DNA模型”向“物种/生态位特化模型”分化。

**2. LLM Agent 与生物信息学深度融合，工具化、平台化趋势明显**
- **GeneX-AI**（30 stars）提出了一个整合LLM多智能体推理与确定性物理/生态求解器的通用计算生物学框架，代表了“LLM Agent + 科学计算”的融合方向。
- **awesome-skills**（4 stars）收集了1800+个用于Claude Code/Codex的Agent技能，涵盖AI4Protein、生信分析等，标志着Bio-AI Agent技能生态正在快速构建。
- **AgentFigureGallery**（112 stars）为编码Agent提供科学绘图技能，属于Agent工具化方向。

**3. 单细胞组学工具链加速向高性能语言（Rust）迁移**
- **scvelo-rs**（12 stars）和 **gffbase**（7 stars）分别用Rust重写了scVelo的RNA velocity计算和GFF3/GTF文件解析引擎，追求CPU上的极致性能。这反映了生信工具“Rust化”的浪潮正在从文件格式处理扩展到核心算法。

**4. 空间组学与比较基因组学方法创新活跃**
- **UniSOR**（8 stars）提出了统一的空间多组学与显微镜数据高保真恢复框架，属于空间组学计算方法。
- **Bioinfo-collinearity-kaks-pipeline**（30 stars）提供了可复现的比较基因组学共线性/KaKs分析流程模板，属于经典生信流程的现代化封装。
- **KaryoScope**（11 stars）提出了面向泛基因组时代的免比对序列注释方法，并附有bioRxiv预印本。

**5. 蛋白质/酶设计方法学创新**
- **CatIF-RL**（4 stars）提出了“活性导向的酶序列设计”，结合逆蛋白质折叠与强化学习，并附有bioRxiv预印本和Zenodo DOI。

**趋势判断**：本期高星项目呈现 **“大模型底座 + Agent工具生态 + 高性能工程化”** 的三层结构。基因组基础模型正在分化出通用型和特化型；LLM Agent正从概念验证走向生物信息学工具的平台化集成；Rust等高性能语言在生信工具链中的地位持续上升。
