# Bio-AI 科研情报雷达 - 2026-06-06

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 5）共 19 个，呈现出以下显著趋势：

### 1. **基因组基础模型进入“多物种 + 微生物”新维度**
- Hugging Face 的 **Carbon**（190 stars）以 1T tokens / 6T DNA 碱基对训练，是当前最大规模的因果 DNA 语言模型之一，且提供了零样本评估套件，标志着基因组基础模型从“单物种”走向“通用基因组”。
- BGI 杭州 AI 团队的 **Genos-m**（19 stars）聚焦人类相关微生物基因组，填补了微生物组基础模型的空白，并附有技术报告（PDF）。
- **dnabert-species**（7 stars）引入物种感知的 DNA 嵌入，支持从序列直接预测物种分类，体现了“物种特异性”与“基础模型”的融合趋势。

### 2. **LLM Agent 技能生态爆发式增长，Bio-AI 成为核心应用场景**
- **awesome-bio-agent-skills**（34 stars）收录 1,676 条生物医学 AI Agent 技能，覆盖基因组学、蛋白质组学、单细胞分析、临床 AI 和蛋白质设计。
- **awesome-genomic-skills**（33 stars）聚焦基因组学与生物信息学的 Agent 技能和 MCP 服务器，面向 Claude Code、Copilot、Cursor 等主流编码 Agent。
- **awesome-skills**（5 stars）收录 1,800+ 条技能，明确标注 AI4Protein 和生物信息学方向。
- **趋势判断**：Bio-AI 正从“模型驱动”转向“Agent 技能驱动”，LLM Agent 工具化、平台化趋势明显，MCP 协议成为连接生物信息学工具与 LLM 的关键桥梁。

### 3. **单细胞分析工具链持续进化：轻量化、加速化、基础模型化**
- **PhenoPixel**（322 stars）提供端到端的单细胞显微镜图像分析 Web 应用，已部署于广岛大学，是典型的工具化平台。
- **nano-scGPT**（9 stars）以 ~270 行纯 PyTorch 代码重新实现 scGPT，嵌入数值等价于原版，运行速度提升 1.5 倍，体现了“轻量化复现 + 可读性优先”的趋势。
- **scvelo-rs**（12 stars）用 Rust + PyO3 加速 scVelo 的 RNA velocity 计算，是“高性能语言 + 生物信息学”的典型代表。
- **Awesome-Single-Cell-Foundation-Models**（8 stars）系统梳理单细胞基础模型资源，说明该领域已形成独立的研究生态。

### 4. **多组学与空间组学走向统一框架**
- **UniSOR**（8 stars）提出统一的空间多组学与显微镜高保真恢复框架，覆盖 MSI、SRT、SRP 等多种技术平台，体现了“跨模态统一建模”的技术方向。

### 5. **可复现流程模板与教学资源持续涌现**
- **Bioinfo-collinearity-kaks-pipeline**（30 stars）提供共线性与 Ka/Ks 分析的标准化流程模板。
- **omics-pipelines**（5 stars）基于 Snakemake + pixi 的组学分析流程集合，覆盖 RNA-seq、ATAC-seq、ChIP-seq 等。
- **genomics-introduction**（5 stars）是 DKFZ 的基因组学教学课件，说明高质量教学资源仍是社区刚需。

**总体判断**：本期高星项目呈现 **“基础模型 + Agent 技能 + 轻量化工具 + 统一框架”** 四轮驱动格局。LLM Agent 化是最大增量方向，基因组基础模型进入多物种/微生物时代，单细胞分析工具加速向 Rust/PyTorch 轻量化演进。
