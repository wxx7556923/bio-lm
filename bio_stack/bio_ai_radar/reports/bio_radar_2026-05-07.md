# Bio-AI 科研情报雷达 - 2026-05-07

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现以下技术趋势：

1. **生物信息学 Agent 化加速**：多个高星项目（如 `bioagent`、`bioinfor-claw`、`bioinformatics-agent-skills`）聚焦于将生物信息学分析流程封装为 AI Agent 或 MCP 服务。`bioagent` 实现了从文献调研到稿件撰写的全自动端到端流程；`bioinformatics-agent-skills` 以 MCP 服务器形式提供 78 个生物信息学工作流的知识图谱查询。这表明**工具化 + LLM Agent 化**正在成为生物信息学自动化的主流范式。

2. **蛋白质结构预测生态持续扩展**：虽然 AlphaFold 3 本身是成熟项目，但围绕其生态的工具链（如 `AF3Parallel` 多 GPU 调度、`AlphaFold3_workflow` Snakemake 流程、`openfold-mac-tool` Apple Silicon 适配）正在快速涌现，方向偏向**平台化和工具化**，而非基础模型创新。

3. **多模态与跨架构蒸馏**：`TIDE` 项目展示了扩散 LLM 的跨架构蒸馏技术，虽非直接生物应用，但其方法论可能影响生物序列模型（如蛋白质语言模型）的知识蒸馏策略。

4. **科研绘图 Agent 化**：`AgentFigureGallery` 为 Claude Code 等编码 Agent 提供科学绘图技能，属于**工具化 + Agent 化**方向，可降低生物信息学结果可视化的门槛。

5. **泛基因组工具链活跃**：多个低星但活跃更新的项目（`panscan`、`Panex_Privus`、`EarlGreyParTEA`、`stoat` 等）聚焦于泛基因组分析，包括结构变异检测、转座元件注释、私有等位基因发现等，方向偏向**工具化**。
