# Bio-AI 科研情报雷达 - 2026-05-19

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 5）共18个，呈现出以下显著技术趋势：

**1. Rust/PyO3 加速生物信息学工具成为主流范式**
- 多个项目采用 Rust + PyO3 重写经典生物信息学工具，追求性能提升和生态整合。`scvelo-rs`（scVelo 的 Rust 加速版）、`rustscenic`（SCENIC 单细胞调控网络分析的 Rust 实现）、`chelae`（NGS 数据修剪的 Rust 工具）、`gffbase`（Rust+DuckDB 的 GFF3/GTF 引擎）和 `poa-consensus`（Partial Order Alignment 的 Rust 库）共同构成了一个强大的 Rust 生物信息学工具链。这表明社区正在从 Python 原型向高性能生产级工具迁移。

**2. LLM Agent 深度渗透计算生物学全流程**
- `GeneX-AI` 将 LLM 多智能体推理与确定性物理/生态求解器整合，形成通用计算生物学框架；`variant-triage` 将 LLM 驱动的变异解读与 ACMG/AMP 规则结合，强调可复现性；`BioinfoFlow` 和 `bioAF` 则用 Agent 思想编排生物信息学管道。趋势是从“LLM 辅助分析”走向“LLM 作为分析流程的核心编排者”。

**3. 单细胞组学工具加速与平台化**
- `scvelo-rs`（RNA velocity 加速）、`rustscenic`（调控网络分析）、`cnvturbo`（CNV 分析，~100× 加速）和 `UniSOR`（空间组学恢复）均聚焦单细胞/空间组学。其中 `UniSOR` 覆盖 MSI、SRT、SRP 等多种空间组学模态，体现了多模态融合趋势。

**4. 作物基因组学基础模型初现**
- `OneGenome-Rice` 是水稻基因组基础模型（1.25B 参数，MoE 架构），可处理长达 1M bp 的 DNA 序列，预训练于 422 个水稻基因组。这是作物基因组学领域少见的公开基础模型项目，值得持续关注。

**5. 工具化与平台化方向分化明显**
- **工具化**：`chelae`、`gffbase`、`poa-consensus`、`cnvturbo` 等专注于单一任务的性能优化。
- **平台化**：`bioAF`（自托管生物信息学平台）、`BioinfoFlow`（Agentic 管道编排系统）提供端到端解决方案。
- **LLM Agent 化**：`GeneX-AI`、`variant-triage` 将 LLM 作为核心推理引擎。
- **多模态化**：`UniSOR` 统一处理空间组学与显微镜数据。
