# Bio-AI 科研情报雷达 - 2026-05-28

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现出以下三大技术趋势：

1.  **基因组基础模型进入“规模化+专业化”双轨阶段**：以 Hugging Face 的 **Carbon**（1T tokens DNA/6T 碱基对）和 BGI 的 **Genos-m**（人类微生物组）为代表，基因组基础模型不再仅追求更大的参数量，而是开始聚焦特定领域（如微生物组）并配套发布标准化评估框架（Carbon 的零样本评估套件）。这表明该领域正从“训练模型”向“建立可复现评估生态”过渡，工具化和平台化特征显著增强。

2.  **AI Agent 技能生态爆发，生物医药成为核心应用场景**：本周涌现了多个专注于“AI Agent 技能”的仓库，如 **awesome-bio-agent-skills**（1,629 个生物医学技能）、**awesome-skills**（1,800+ 技能，涵盖 AI4Protein）和 **AgentFigureGallery**（科学绘图技能）。这些项目并非独立的生物信息学工具，而是为 Claude Code、Codex 等通用编码 Agent 提供“即插即用”的领域知识包。这标志着 Bio-AI 正在从“专用模型”向“LLM Agent + 领域技能库”的范式转变，**LLM Agent 化** 和 **多模态化**（如科学绘图）是核心方向。

3.  **高性能计算与生信流程的“Rust 化”加速**：**scvelo-rs**（单细胞 RNA velocity 的 Rust 加速）和 **gffbase**（基于 Rust+DuckDB 的 GFF3/GTF 引擎）表明，社区正在用 Rust 重写生物信息学中的性能瓶颈组件，并通过 PyO3 提供无缝的 Python 接口。这是一种典型的 **工具化** 趋势，旨在不改变用户使用习惯的前提下，大幅提升计算效率。
