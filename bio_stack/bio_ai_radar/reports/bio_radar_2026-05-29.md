# Bio-AI 科研情报雷达 - 2026-05-29

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现出以下显著趋势：

1.  **基因组基础模型进入“规模化+专业化”阶段**：以 Hugging Face 的 **Carbon** 为代表，基因组基础模型已从概念验证走向大规模预训练（1T tokens DNA），并开始提供标准化的零样本评估套件。同时，针对特定领域的模型开始涌现，如 BGI 的 **Genos-m** 专注于人类微生物组，**dnabert-species** 则引入了物种感知能力。这表明该领域正从通用模型向“通用底座 + 领域专用”的生态演进。

2.  **AI Agent 技能生态爆发，工具化与平台化趋势明显**：本周出现了多个专注于收集、整理和分发 AI Agent 技能的仓库，如 **awesome-bio-agent-skills**（1629+ 技能）和 **awesome-skills**（1800+ 技能）。这些项目并非开发新的模型，而是构建一个“技能市场”或“技能索引”，让生物学家能通过自然语言指令（如 Claude Code、Codex）直接调用复杂的生物信息学分析流程。这标志着 Bio-AI 正从“模型驱动”向“Agent 驱动”的工具化、平台化方向快速演进。

3.  **LLM Agent 与确定性科学计算深度融合**：**GeneX-AI** 项目提出了一个通用计算生物学框架，将 LLM 多智能体的推理能力与物理、生态等确定性求解器相结合。这种“LLM Agent + 传统科学计算”的混合范式，旨在利用 LLM 进行高层次的任务规划与设计，同时依赖传统工具进行精确计算，代表了 AI for Science 的一个重要发展方向。

4.  **高性能计算与生物信息学工具“Rust 化”**：**scvelo-rs** 和 **gffbase** 两个项目分别使用 Rust 重写了单细胞 RNA 速率分析（scVelo）和 GFF/GTF 注释文件处理引擎。这表明社区正在通过 Rust 等高性能语言来加速生物信息学核心工具，以应对日益增长的数据规模，体现了工具优化的趋势。

5.  **泛基因组时代的新工具涌现**：**KaryoScope** 项目针对泛基因组时代，提出了无需比对的快速序列注释方法，并已发布 bioRxiv 预印本。这反映了随着泛基因组参考的普及，对新型、高效分析工具的需求正在增长。
