# Bio-AI 科研情报雷达 - 2026-05-11

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 5）共 20 个，呈现出以下显著趋势：

**1. Agent 化全面渗透生物信息学各环节**
- 从文献综述到论文撰写（`bioagent`）、变异解读（`variant-triage`、`agentic-genomics`）、细胞类型注释（`celltype-agent`）、GEO 数据集筛选（`GEO_Screener`），再到发育生物学评估（`BRIDGE`），LLM Agent 已覆盖生信全流程。
- 趋势判断：**工具化 + Agent 化** 是当前最活跃的方向，且正从“单 Agent 演示”向“多 Agent 协作系统”演进（如 `bioagent` 的端到端零人工干预设计）。

**2. 科学绘图与编码 Agent 基础设施成熟**
- `AgentFigureGallery` 为 Claude Code、Codex 等编码 Agent 提供“即插即用”的科学绘图技能，标志着 Agent 生态开始出现**平台化、模块化**的基础设施层。
- `agentic-coding-intro` 则从教育角度系统化 Agent 编码方法论，说明该领域已进入知识沉淀阶段。

**3. Rust 重写浪潮加速——性能导向的工具化**
- 多个项目选择 Rust + PyO3 重写经典生信工具：`rustscenic`（SCENIC+ 调控网络）、`scvelo-rs`（RNA velocity）、`vcfkit`（VCF 处理）、`gffbase`（GFF3/GTF 引擎）、`poa-consensus`（部分序比对）。
- 趋势判断：**工具化 + 性能优化** 是 Rust 重写的核心驱动力，目标是一键安装、零 Java 依赖、低内存、高吞吐。这反映了生信社区对“现代工具链”的强烈需求。

**4. 单细胞与空间组学仍是热点，但更聚焦于特定痛点**
- `scnoisemeter`（技术噪声量化）、`cnvturbo`（CNV 分析加速）、`sc-Sigflow`（细胞通讯分析）、`celltype-agent`（自动注释）——均针对单细胞/空间组学流程中的具体瓶颈，而非泛化工具。
- 趋势判断：单细胞领域正从“能做”转向“做准、做快、可解释”。

**5. 作物基因组基础模型出现**
- `OneGenome-Rice` 是水稻专用的基因组基础模型（1.25B 参数，MoE 架构），支持百万碱基级序列处理。这是**作物基因组 + 基础模型**方向的重要信号，可能带动更多作物专用基因组语言模型出现。

**6. 本地化、隐私优先的 AI 药物发现**
- `PharmaCore` 强调“完全本地运行、无需云、无需 GPU”，面向 Apple Silicon 的 AI 药物发现。这反映了对**数据主权和可审计性**的日益关注。
