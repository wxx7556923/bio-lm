# Bio-AI 科研情报雷达 - 2026-06-02

## 一、Top 高星区技术风向

本期高星区项目（stars ≥ 5）呈现出以下几个显著的技术趋势：

1.  **基因组基础模型进入“实用化”与“专业化”阶段**：以 Hugging Face 的 `carbon`（188 stars）和 BGI 的 `Genos-m`（17 stars）为代表，基因组基础模型不再仅停留在论文层面，而是开始提供完整的评估代码、微调脚本和预训练权重。`carbon` 强调零样本评估的可复现性，`Genos-m` 则专注于人类微生物组这一特定领域，显示出模型正从通用型向领域专用型分化。

2.  **AI Agent 技能生态爆发式增长**：`awesome-bio-agent-skills`（29 stars）、`awesome-genomic-skills`（22 stars）和 `awesome-skills`（5 stars）三个项目均致力于收集和整理用于生物医学研究的 AI Agent 技能（Skills）和 MCP 服务器。这标志着 Bio-AI 领域正从“开发模型”转向“构建可复用的 Agent 工作流”，工具化和平台化趋势明显。这些项目本身是“元仓库”（curated list），但反映了社区对 Agent 化生物信息学分析的高度热情。

3.  **单细胞分析工具链加速与平台化**：`PhenoPixel`（338 stars）提供了一个端到端的单细胞显微镜图像分析 Web 应用，`scvelo-rs`（12 stars）则通过 Rust + PyO3 对经典的 RNA velocity 工具 scVelo 进行加速。这表明单细胞分析领域一方面在向用户友好的平台化发展，另一方面在底层计算效率上进行优化。

4.  **LLM Agent 与物理/生态求解器的融合**：`GeneX-AI`（30 stars）提出了一个将 LLM 多智能体推理与确定性物理和生态求解器相结合的通用计算生物学框架。这代表了 Bio-AI 的一个前沿方向：利用 LLM 的推理能力来编排和调用传统的计算工具，实现更复杂的自动化科学发现流程。

5.  **泛基因组时代的新工具涌现**：`KaryoScope`（13 stars）专注于为泛基因组时代提供快速、无需比对的序列注释工具，并已发布 bioRxiv 预印本。这反映了随着泛基因组参考的普及，对新型比对和注释算法的需求正在增长。

**总结**：本期高星项目呈现出“模型+工具+Agent”三位一体的发展态势。基因组基础模型走向实用，AI Agent 技能生态快速构建，单细胞分析工具持续进化，而 LLM Agent 与传统计算方法的融合则代表了更具前瞻性的探索。
