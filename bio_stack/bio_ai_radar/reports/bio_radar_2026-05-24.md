# Bio-AI 科研情报雷达 - 2026-05-24

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现出以下三大技术趋势：

1.  **基因组基础模型加速“物种化”与“微生物化”：** 以 Hugging Face 的 **Carbon** 和 BGI 杭州的 **Genos-m** 为代表，基因组基础模型不再局限于单一模式（如人类），而是开始向**特定物种（如人类微生物组）** 和**大规模多物种**方向分化。Carbon 在 1T tokens 的 DNA/RNA 数据上训练，而 Genos-m 则专注于人类相关微生物。这表明该领域正从通用模型向更精细、更专业的应用场景演进。

2.  **单细胞与空间组学分析工具“工程化”与“加速化”：** 多个项目聚焦于提升经典生信工具的性能和易用性。**scvelo-rs** 通过 Rust + PyO3 将单细胞 RNA velocity 分析工具 scVelo 加速，**cnvturbo** 则将 R 语言 inferCNV 重写为 Python 并实现百倍加速，且深度集成 Scanpy 生态。**UniSOR** 则提出了一个统一框架用于空间多组学和显微成像的高保真恢复。这反映了社区对**高性能、可复现、生态兼容**的工具化需求日益强烈。

3.  **AI Agent 与 LLM 框架向“垂直领域”渗透：** **GeneX-AI** 是一个典型的 LLM 多智能体框架，但专门为计算生物学设计，集成了物理/生态求解器。**AgentFigureGallery** 则是一个为科学绘图设计的 Agent 技能包。这表明 LLM Agent 技术正从通用平台向**生物信息学、科学可视化等垂直领域**进行深度定制和工具化，旨在降低科研人员使用 AI 的门槛。

**趋势判断：** 当前高星项目更偏向**工具化**和**平台化**，旨在解决具体、可量化的计算瓶颈（如速度、可复现性）。同时，**LLM Agent 化**和**多模态化**（如空间组学+显微成像）的趋势也已显现，但尚处于早期探索阶段。
