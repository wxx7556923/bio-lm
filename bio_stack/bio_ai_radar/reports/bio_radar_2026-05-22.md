# Bio-AI 科研情报雷达 - 2026-05-22

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 5）呈现出以下三大技术趋势：

1.  **基因组基础模型进入“专业化+可部署”阶段**：以 Hugging Face 的 **Carbon**（125 stars）和 BGI 的 **Genos-m**（6 stars）为代表，基因组基础模型不再只是论文中的概念验证，而是开始提供完整的预训练权重、评估代码和微调脚本。Carbon 主打通用 DNA/RNA 语言模型（1T tokens），Genos-m 则聚焦人类微生物组，体现了从“通用”到“领域专用”的分化趋势。**平台化、工具化**特征明显。

2.  **LLM Agent 深度渗透计算生物学工作流**：**GeneX-AI**（30 stars）将 LLM 多智能体推理与物理/生态求解器结合，构建通用计算生物学框架；**BioinfoFlow**（4 stars）和 **bioAF**（5 stars）则分别从 Agentic 管道编排和平台化基础设施切入。这标志着 Bio-AI 正从“单一模型预测”向“多智能体协作 + 自动化工作流”演进，**LLM Agent 化**趋势显著。

3.  **高性能 Rust/Python 加速工具成为新基建**：**scvelo-rs**（12 stars）、**gffbase**（7 stars）、**poa-consensus**（6 stars）和 **cnvturbo**（6 stars）均采用 Rust 或纯 Python 重写经典生物信息学工具（scVelo、GFF3/GTF 解析、Partial Order Alignment、inferCNV），追求 10-100 倍加速和与现有生态（Scanpy/AnnData）的无缝集成。这反映了社区对**高性能、可复现、生态兼容**的基础工具的强烈需求，**工具化**趋势突出。

**趋势判断**：本期高星项目呈现出“大模型（基因组基础模型）+ 小工具（高性能加速）+ 智能体编排（LLM Agent）”的三层技术栈雏形。基因组基础模型正在从“能跑”走向“好用”，而 LLM Agent 正在将分散的工具串联成自动化工作流。
