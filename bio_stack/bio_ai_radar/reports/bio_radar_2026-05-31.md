# Bio-AI 科研情报雷达 - 2026-05-31

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现三大技术趋势：

1. **基因组基础模型进入“规模化+专业化”阶段**：以 Hugging Face 的 `carbon`（182 stars）和 BGI 的 `Genos-m`（16 stars）为代表，基因组基础模型不再仅追求更大的参数量，而是向**特定物种/领域**（如人类微生物组）和**可复现评估体系**（如 Carbon 提供的零样本评估套件）发展。模型权重和训练数据均已公开，标志着该领域从“论文附属代码”向“可复现科研基础设施”转变。

2. **AI Agent 技能生态爆发，Bio-Agent 成为独立赛道**：`awesome-bio-agent-skills`（25 stars）、`awesome-genomic-skills`（18 stars）、`awesome-skills`（5 stars）等“awesome”系列项目集中涌现，专门为 Claude Code、Codex、Cursor 等编码 Agent 提供生物信息学技能（Markdown 指令 + MCP 服务器）。这标志着 **Bio-AI Agent 正在从概念验证走向工具化、平台化**，形成了“技能市场”式的生态。`GeneX-AI`（30 stars）更进一步，将 LLM 多智能体推理与物理/生态求解器结合，体现了 **LLM Agent + 确定性计算**的混合架构趋势。

3. **高性能计算（Rust/PyO3）加速生物信息学工具**：`scvelo-rs`（12 stars）和 `gffbase`（7 stars）均采用 Rust + PyO3 技术栈，对单细胞 RNA 速率分析和 GFF/GTF 注释文件处理进行加速。这反映了生物信息学社区对**计算效率**的迫切需求，以及 Rust 在科学计算领域渗透的加速趋势。`KaryoScope`（13 stars）则针对泛基因组时代提出无比对序列注释方法，体现了**泛基因组工具链**的早期布局。

**趋势判断**：当前 Bio-AI 高星项目更偏向**工具化**（可复现流程、高性能库）和**平台化**（Agent 技能生态、基础模型评估框架），LLM Agent 化正在从通用 Agent 向**领域专用 Agent**（生物信息学 Agent）演进，多模态化在本次高星项目中表现不明显。
