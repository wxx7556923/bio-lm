# Bio-AI 科研情报雷达 - 2026-05-20

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 20）共 7 个，呈现出以下技术趋势：

1.  **基因组基础模型进入“物种专用 + 开源评估”阶段**：Hugging Face 的 **Carbon** 项目（60 stars）是一个面向 DNA/RNA 的因果语言模型，训练于 1T tokens / 6T 碱基对，并提供了零样本评估代码。与此同时，**OneGenome-Rice**（6 stars）则聚焦水稻基因组，采用 MoE 架构处理百万级长度序列。这表明基因组基础模型正从通用预训练向“物种专用 + 可复现评估”方向分化。

2.  **LLM Agent 深度嵌入计算生物学工具链**：**GeneX-AI**（24 stars）提出了一个“LLM 多智能体 + 物理/生态求解器”的通用计算生物学框架，内部包含架构师、理论物理学家、评审委员会等 Agent 角色。**variant-triage**（20 stars）则将 LLM 用于临床变异解读，但强调通过 ACMG/AMP 规则约束生成可审计、可复现的结果。这显示 Agent 不再只是聊天界面，而是开始作为“可编程的推理层”嵌入生物信息学流程。

3.  **高性能 Rust 重写浪潮持续**：**scvelo-rs**（12 stars）和 **gffbase**（7 stars）分别用 Rust + PyO3 重写了 scVelo 和 GFF3/GTF 解析引擎，追求数量级加速。**chelae**（7 stars）也是 Rust 实现的 NGS 数据修剪工具。Rust 正在成为生物信息学高性能基础设施的首选语言。

4.  **工具化与平台化趋势明显**：**AgentFigureGallery**（107 stars）为编码 Agent 提供科学绘图技能，属于“Agent 技能市场”类工具。**bioAF**（5 stars）和 **BioinfoFlow**（4 stars）则分别提供自托管生物信息学平台和 Agent 驱动的流程编排系统，体现了平台化、低代码化的方向。

**趋势判断**：本期高星项目整体偏向 **工具化** 和 **平台化**，LLM Agent 的应用正在从“对话式”转向“嵌入式推理层”，而 Rust 重写则代表了性能优化的主流路径。多模态（如空间组学 + 显微镜）和 LLM Agent 化在低星项目中已有萌芽。
