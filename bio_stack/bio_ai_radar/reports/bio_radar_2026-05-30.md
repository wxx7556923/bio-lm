# Bio-AI 科研情报雷达 - 2026-05-30

## 一、Top 高星区技术风向

本期高星项目呈现出几个鲜明的技术趋势：

1.  **基因组基础模型进入“工业化”阶段**：以 Hugging Face 的 **Carbon** 和 BGI 的 **Genos-m** 为代表，基因组基础模型不再仅仅是研究论文中的概念，而是开始提供标准化的预训练权重、评估基准和微调脚本。Carbon 强调“零样本评估”的可复现性，Genos-m 则专注于人类微生物组这一垂直领域。这表明该领域正从“造模型”转向“用模型”，工具化和平台化趋势明显。

2.  **AI Agent 技能生态爆发，LLM Agent 化成为主流范式**：本期出现了多个专注于“Bio-Agent Skills”的仓库（如 `awesome-bio-agent-skills`、`awesome-genomic-skills`、`awesome-skills`），它们将生物信息学任务（如序列分析、蛋白质设计）封装成可供 Claude Code、Codex 等编码 Agent 调用的“技能”或 MCP 服务器。这标志着生物信息学正快速拥抱 LLM Agent 范式，通过自然语言驱动复杂分析流程，降低使用门槛。

3.  **多模态与跨尺度融合**：`UniSOR` 项目致力于统一处理空间分辨多组学（如 MSI、SRT）和显微镜数据，体现了从单一组学向多模态、跨尺度数据融合的趋势。`GeneX-AI` 则更进一步，试图将 LLM 推理与物理/生态求解器结合，代表了“AI for Science”中多智能体协作解决复杂科学问题的方向。

4.  **高性能计算与工程优化**：`scvelo-rs` 和 `gffbase` 等项目使用 Rust 重写核心生物信息学工具（单细胞 RNA 速率、GFF/GTF 解析），追求极致的性能和内存效率。这表明在数据量爆炸的背景下，底层工程的优化成为提升科研效率的关键。

**总结**：本期高星项目清晰地展示了 Bio-AI 领域正在从“模型创新”向“平台化、工具化、Agent 化”演进。基因组基础模型趋于成熟，AI Agent 正在重塑生物信息学的工作流，而多模态融合和高性能工程则成为新的技术高地。

## 二、最新更新区低星潜力项目深挖

本期“最新更新”区项目多为 LLM 推理、Agent 框架和评估工具，与 Bio-AI 直接相关的项目较少。但以下项目值得关注：

### 值得跟踪的项目清单

1.  **bio-sae** (`jascal/bio-sae`)
    -   **理由**：该项目明确将蛋白质语言模型（ESM-2）的激活值打包成张量束，并附带丰富的生物学真值标签（GO 术语、Pfam 域、EC 编号等），用于 SAE（稀疏自编码器）的可解释性研究。这是一个非常前沿的方向，旨在理解蛋白质语言模型“学到了什么”。项目提供了标准化的基准测试平台，对于推动 AI 蛋白质科学的可解释性研究具有重要价值。
    -   **线索**：项目描述和 README 明确提到了 ESM-2、SAE、GO、Pfam 等关键术语，是一个高质量的研究基准项目。

2.  **cplm** (`yyoshikai/cplm`)
    -   **理由**：项目名称为“chemical-protein language model”，直接指向化学-蛋白质相互作用这一核心药物发现领域。虽然目前处于“under construction”状态，且 README 内容较为初步，但其研究方向明确，且由学术团队（Mizuno group）维护。一旦正式发布，可能是一个重要的分子-蛋白交叉模型。
    -   **线索**：项目描述和 README 提到了“BindGPT”和“chemical-protein language model”，但尚未提供论文或模型权重链接，需要进一步核验。

### 其他值得关注的项目（与 Bio-AI 间接相关）

-   **KVBoost** (`pythongiant/KVBoost`)：虽然是一个通用的 LLM 推理加速工具，但其 KV 缓存复用技术对于处理长序列的生物序列（如长 DNA 序列、蛋白质序列）的 LLM 推理具有潜在价值。
-   **CausalAgentBench** (`brycewang-stanford/CausalAgentBench`)：该项目旨在构建一个用于评估 LLM Agent 因果推断能力的基准。虽然目前内容较少，但“因果推断”在生物医学（如药物靶点发现、基因调控网络推断）中至关重要，该基准的发布可能对 Bio-AI Agent 的评估产生深远影响。
