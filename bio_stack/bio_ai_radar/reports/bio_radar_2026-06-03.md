# Bio-AI 科研情报雷达 - 2026-06-03

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现出以下三大技术趋势：

1.  **基因组基础模型进入“物种感知”与“微生物专精”阶段**：以 Hugging Face 的 **Carbon** 和 BGI 的 **Genos-m** 为代表，基因组基础模型不再局限于通用 DNA 序列，而是开始向**特定生态位**（如人类微生物组）和**物种感知能力**（如 dnabert-species）分化。这表明该领域正从“大一统”预训练转向**领域特化与下游任务对齐**，更强调模型在具体生物学场景（如微生物组研究、物种分类）中的实用性。

2.  **AI Agent 技能生态爆发，推动“Bio-Agent-as-a-Service”**：`awesome-bio-agent-skills`、`awesome-genomic-skills` 和 `awesome-skills` 三个项目在短时间内获得高星，标志着生物信息学领域正在系统性地构建 **AI Agent 的技能库和 MCP（Model Context Protocol）服务器**。这不再是单个工具，而是**平台化、生态化**的 Agent 技能集市，旨在让 Claude Code、Cursor 等通用编码 Agent 能够直接调用生物信息学专业工具（如基因组分析、蛋白质设计），推动“Bio-Agent”从概念走向工程落地。

3.  **单细胞与空间组学分析工具加速“工程化”与“加速化”**：`PhenoPixel` 提供了端到端的单细胞显微镜图像分析 Web 应用，`scvelo-rs` 通过 Rust 重写加速 RNA 速度分析，`UniSOR` 则试图统一空间多组学与显微镜数据的恢复框架。这些项目表明，单细胞和空间组学领域正从算法研究转向**高性能、可部署的工程化工具**，强调用户体验、计算效率和跨模态整合。

**趋势判断**：当前高星项目更偏向**工具化**（PhenoPixel, scvelo-rs）和**平台化/生态化**（Carbon, awesome-bio-agent-skills）。LLM Agent 化是核心驱动力，但多模态化（如 UniSOR 整合空间组学与显微镜）尚处于早期探索阶段。

## 二、最新更新区低星潜力项目深挖

在最新更新区中，以下低星项目因包含论文、模型或可复现实验线索，值得重点关注：

### 值得跟踪的项目清单

1.  **chimerax-vampnet** (`m9h/chimerax-vampnet`)
    -   **理由**：该项目将 VAMPnet（马尔可夫状态模型）集成到 UCSF ChimeraX 中，用于分析蛋白质构象景观。它支持多源系综（MD、AlphaFlow、BioEmu），并提供了 MCP 桥接供 LLM Agent 驱动。项目描述中明确引用了 **Mardt et al. 2018** 的 VAMPnet 论文，并提供了可复现的测试（18/18 测试通过）。这是一个将**经典计算化学方法（MSM）与现代 AI 系综生成（AlphaFlow）及 LLM Agent 结合**的创新尝试，具有很高的学术和工程跟踪价值。

2.  **CatIF-RL** (`lynk-101-li/CatIF-RL`)
    -   **理由**：该项目专注于“活性导向的酶序列设计”，通过逆向蛋白质折叠和强化学习实现。README 中明确提供了 **bioRxiv 预印本链接**（DOI: 10.64898/2026.05.11.724288）和 **Zenodo 代码存档**（DOI: 10.5281/zenodo.20357062）。这直接对应一篇新论文，且代码、模型和数据（通过 Zenodo）均可复现，是典型的**新论文+可复现代码**项目。

3.  **KaryoScope** (`barthel-lab/KaryoScope`)
    -   **理由**：该项目旨在为泛基因组时代提供快速、免比对的序列注释工具。README 中明确提供了 **bioRxiv 预印本链接**（DOI: 10.64898/2026.05.15.725544）。这是一个**新工具+新论文**的组合，专注于泛基因组分析这一前沿领域，具有重要的工具价值。

4.  **REFOLD** (`aesenthilvanan-coder/REFOLD`)
    -   **理由**：该项目声称是“世界首个蛋白质组规模的药理伴侣数据库”，是一个端到端的计算流程，用于预测疾病相关错义突变是否导致蛋白质错误折叠，并设计小分子候选药物。虽然目前星数较低，但其描述涉及**蛋白质设计、药物发现和计算生物学**的交叉领域，且提供了完整的流程描述，可能对应一篇即将发表或已发表的论文（需要进一步核验）。

5.  **Genos-m** (`BGI-HangzhouAI/Genos-m`)
    -   **理由**：虽然该项目在“高星”区，但其星数（17）相对较低，且更新非常新。它提供了**Hugging Face 模型权重**和一份**技术报告（PDF）**。这是一个由 BGI 发布的、针对人类微生物组的基因组基础模型，直接提供了模型和报告，是**新模型+新数据**的典型代表。
