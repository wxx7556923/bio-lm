# Bio-AI 科研情报雷达 - 2026-05-09

## 一、Top 高星区技术风向

本周高星项目呈现三大技术趋势：

**1. LLM Agent 全面渗透生物信息学工具链（工具化 + Agent 化）**
高星项目中，Agent 类项目占比极高，且已从概念验证进入实用阶段。`bioagent`（97 stars）提出端到端自主生物信息学研究系统，`bioinformatics-agent-skills`（48 stars）将 78 个生信工作流封装为 MCP 服务器供 Claude Code 调用，`celltype-agent`（12 stars）专注单细胞注释，`variant-triage`（19 stars）和 `agentic-genomics`（6 stars）聚焦临床变异解读。趋势判断：**生物信息学正从“手动写脚本”转向“Agent 编排工作流”，MCP 协议成为关键基础设施。**

**2. 单细胞与空间组学工具持续活跃，但更强调跨平台兼容性（工具化）**
`scnoisemeter`（12 stars）支持 ONT、PacBio、短读等多种平台的技术噪声量化，`rustscenic`（10 stars）用 Rust 重写 SCENIC+ 管线，消除 Java 依赖，`sc-Sigflow`（9 stars）专注细胞通讯分析。趋势判断：**单细胞工具正从“单一平台”向“跨平台通用”演进，性能优化（Rust 重写）成为新方向。**

**3. 基因组基础模型与作物基因组学出现新玩家（平台化）**
`OneGenome-Rice`（6 stars）是 BGI 推出的水稻基因组基础模型（1.25B 参数，MoE 架构），支持百万碱基长度输入。这是继 Evo、DNABERT 之后，**首个针对作物基因组的大规模基础模型**，标志着基因组语言模型从人类/模式生物向农业基因组扩展。

**4. 科学绘图 Agent 技能成为新品类（多模态化）**
`AgentFigureGallery`（104 stars）为 Claude Code、Codex 等编码 Agent 提供科学绘图技能，支持视觉参考 + 人类反馈驱动的绘图指导。这代表了 **Bio-AI 从“分析”向“可视化输出”延伸**的趋势。
