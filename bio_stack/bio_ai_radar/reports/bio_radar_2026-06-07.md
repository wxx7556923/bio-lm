# Bio-AI 科研情报雷达 - 2026-06-07

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 5）呈现出以下显著趋势：

1.  **基因组基础模型进入“实用化”与“专业化”阶段**：以 Hugging Face 的 **Carbon** 和 BGI 的 **Genos-m** 为代表，基因组大模型不再仅停留在论文层面，而是开始提供完整的模型权重、评估代码和微调脚本。Carbon 主打通用 DNA 大模型（1T tokens），而 Genos-m 则聚焦于人类微生物组，体现了从“通用”到“领域专用”的垂直分化趋势。

2.  **AI Agent 技能生态爆发，生物信息学成为核心应用场景**：本期出现了多个专注于“生物 AI Agent 技能”的 curated list 项目（如 `awesome-bio-agent-skills`、`awesome-genomic-skills`、`awesome-skills`），且星数增长迅速。这表明社区正在系统性地为 Claude Code、Copilot 等 AI 编码助手构建生物信息学领域的“技能包”和 MCP 服务器，推动生信分析从“手动写脚本”向“Agent 辅助自动化”转变。这是一个强烈的平台化和生态化信号。

3.  **单细胞分析工具链持续“轻量化”与“加速化”**：`nano-scGPT` 通过纯 PyTorch 重写，将 scGPT 的推理速度提升 10 倍以上，代码仅 270 行，降低了单细胞基础模型的使用门槛。`scvelo-rs` 则通过 Rust + PyO3 对经典的 RNA velocity 工具进行加速。这反映了社区对高性能、易集成、低依赖工具的需求。

4.  **多模态与空间组学分析框架趋于统一**：`UniSOR` 提出了一个统一框架，用于处理空间分辨多组学和显微镜数据的高保真恢复，覆盖 MSI、SRT、SRP 等多种技术。这表明空间组学领域正在从“各自为战”的工具开发，走向“统一范式”的框架整合。

5.  **计算生物学自动化平台（Platform）初现雏形**：`bioAF` 和 `GeneX-AI` 代表了两种不同的平台化思路。`bioAF` 是一个面向小团队的、可自托管的生信管道编排平台，强调基础设施管理。`GeneX-AI` 则是一个将 LLM 多智能体推理与物理/生态求解器结合的通用计算生物学框架，更具“AI-Native”特征。

**趋势判断**：本期高星项目整体偏向 **工具化**（如 scvelo-rs、nano-scGPT）和 **平台化/生态化**（如 Carbon、awesome-bio-agent-skills、bioAF）。LLM Agent 化趋势在“技能”类项目中非常突出，而多模态化则在空间组学（UniSOR）和质谱（OpenTFRaw）领域有所体现。
