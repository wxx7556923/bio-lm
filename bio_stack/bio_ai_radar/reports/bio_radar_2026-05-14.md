# Bio-AI 科研情报雷达 - 2026-05-14

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 5）共 18 个，呈现出以下显著趋势：

**1. Agent 化浪潮全面渗透生物信息学**
- 多个项目将 LLM Agent 模式直接嵌入核心生信流程：`bioagent`（端到端自主研究）、`celltype-agent`（细胞类型注释）、`variant-triage`（临床变异解读）、`agentic-genomics`（变异解释 Agent）。这不再是“套壳”工具，而是将 Agent 作为流程编排的核心引擎。
- 趋势判断：**LLM Agent 正在从“辅助工具”进化为“流程执行者”**，尤其在需要多步推理、多数据库查询的临床基因组学和单细胞分析场景中落地最快。

**2. 高性能 Rust 重写成为“降本增效”主流策略**
- `scvelo-rs`（RNA velocity）、`rustscenic`（单细胞调控网络）、`vcfkit`（VCF 处理）、`gffbase`（GFF/GTF 引擎）、`poa-consensus`（序列比对）—— 五个项目均采用 Rust + PyO3 策略，目标是“pip install 替代 Java/MACS2/dask 等复杂依赖”。
- 趋势判断：**生信工具正在经历“Rust 化”重构**，核心诉求是降低部署复杂度、提升单机 CPU 性能、消除对 Java 和 CUDA 的依赖。这波浪潮对标的是 2015-2020 年的 Python 化浪潮。

**3. 平台化与基础设施化方向明确**
- `bioAF` 提供面向小型生物技术公司的全栈计算平台（Kubernetes + Jupyter + 实验追踪），`bioinformatics-genomic-pipeline` 是 Airflow 编排的生产级流程。这些项目强调“开箱即用”和“可复现”。
- 趋势判断：**生信平台正在从“脚本集合”向“云原生 SaaS”演进**，但当前仍以自托管为主。

**4. 作物基因组学与基础模型开始出现**
- `OneGenome-Rice` 是水稻基因组基础模型（1.25B 参数，MoE 架构），预训练于 422 个水稻基因组。这是本期唯一的大规模基因组语言模型项目，代表 AI for Crop Science 的早期探索。
- 趋势判断：**基因组基础模型正在从人类/模式生物向作物扩展**，但数据量和模型规模仍远小于 ESM2 等蛋白质模型。

**5. 多模态与工具化趋势较弱**
- 本期高星项目以单模态（基因组序列、单细胞表达）为主，未见明显的多模态融合（如基因组+影像+临床文本）。`AgentFigureGallery` 是唯一的“科学绘图 Agent 技能”，属于工具化方向。

### 深度分析

**interpretable-missense-classification** 和 **LM-cancer-detection** 是本期最值得关注的 Bio-AI 论文关联项目。前者直接比较传统序列特征与蛋白质语言模型（pLMs）嵌入，这是当前变异致病性预测领域的热点问题；后者将 DNA 语言模型应用于微生物组癌症检测，属于“AI for Microbiome”交叉方向。两个项目均处于早期阶段（stars=0），但研究设计完整，建议持续跟踪其 arXiv/bioRxiv 预印本发布。

**indic-eval** 虽然聚焦 LLM 评估而非直接生信，但其“可复现审计”的设计理念和 HuggingFace 数据集发布模式，对生信领域的模型评估基准建设有参考意义。
