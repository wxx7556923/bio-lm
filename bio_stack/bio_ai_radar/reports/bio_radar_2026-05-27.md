# Bio-AI 科研情报雷达 - 2026-05-27

## 一、Top 高星区技术风向

近期高星 Bio-AI 项目呈现出以下技术趋势：

1.  **基因组基础模型进入“物种级”与“微生物级”分化阶段**：以 Hugging Face 的 **Carbon** 和 BGI 的 **Genos-m** 为代表，基因组基础模型不再局限于通用人类基因组，而是开始向**特定生态位**（如人类微生物组）和**物种感知**（如 dnabert-species）方向演进。这表明领域正在从“大一统”预训练转向**专业化、场景化**的模型部署。

2.  **LLM Agent 技能生态爆发，Bio-AI 成为核心应用场景**：多个高星项目（如 `AgentFigureGallery`、`awesome-bio-agent-skills`、`awesome-skills`）专注于为 Claude Code、Codex 等编码 Agent 提供**即插即用的科学绘图、生物信息学分析技能**。这标志着 Bio-AI 正从“模型即服务”向“**Agent 技能即服务**”转变，工具化和平台化趋势明显，旨在降低生物学家使用 AI 的门槛。

3.  **计算生物学流程的“可复现性”与“工程化”成为共识**：`Bioinfo-collinearity-kaks-pipeline` 和 `bioAF` 等项目强调**可复现的流程模板**和**自托管的平台化部署**。这反映了社区对标准化、可审计、可共享的计算工作流的需求日益增长，是 Bio-AI 走向成熟和工业级应用的必要条件。

4.  **多模态与空间组学结合，追求高保真度恢复**：`UniSOR` 项目专注于从空间分辨多组学和显微镜数据中进行高保真恢复，体现了 AI 在**整合不同模态数据**（如转录组、蛋白组、代谢组与空间位置信息）方面的前沿探索，是单细胞和空间生物学领域的核心挑战。

**趋势判断**：当前高星项目更偏向**工具化**（Agent 技能、流程模板）和**平台化**（自托管平台），同时**LLM Agent 化**（将 LLM 作为生物信息学任务的编排核心）的趋势非常显著。多模态化在空间组学领域有突出体现，但尚未成为主流。
