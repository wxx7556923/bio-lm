# Bio-AI 科研情报雷达 - 2026-05-16

## 一、Top 高星区技术风向

本期高星项目（stars ≥ 5）共 20 个，呈现出以下显著技术趋势：

1.  **Rust 重写生物信息学核心工具成为主流趋势**：本期高星区最突出的现象是多个项目采用 **Rust + PyO3** 技术栈，对经典生物信息学工具进行高性能重写。例如 `scvelo-rs`（单细胞 RNA velocity）、`rustscenic`（SCENIC+ 调控网络）、`gffbase`（GFF3/GTF 引擎）、`vcfkit`（VCF 工具集）、`poa-consensus`（部分序比对）和 `chelae`（NGS 读段修剪）。这些项目普遍宣称 **“无 Java、无 Python 依赖、单二进制部署、数倍加速”**，标志着生物信息学基础设施正从 Python/R 脚本生态向系统级语言迁移，**工具化、工程化**特征明显。

2.  **LLM Agent 深度渗透基因组学与临床分析**：多个项目将 LLM Agent 应用于具体的基因组学任务，而非泛化讨论。`variant-triage` 将 ACMG/AMP 规则与 LLM 推理结合，实现可审计的变异解读；`celltype-agent` 利用 Claude 自动注释单细胞和空间转录组细胞类型；`agentic-genomics` 使用 LangGraph 构建可追溯推理链的变异解释器。这些项目强调 **“结构化推理 + 可复现性”**，而非简单的“黑箱”生成，体现了 **LLM Agent 化** 在精准医学中的务实落地。

3.  **基因组基础模型向作物领域拓展**：`OneGenome-Rice` 是 BGI 发布的水稻基因组基础模型，采用 MoE 架构、1.25B 参数、支持百万碱基长度，预训练于 422 个水稻基因组。这表明基因组大语言模型正从人类/模式生物向 **作物基因组学** 延伸，**多模态化**（序列 + 表型）和 **平台化**（育种决策支持）是潜在方向。

4.  **AI 药物发现本地化、轻量化**：`PharmaCore` 提出“完全在 MacBook 上运行、无需云端、无需 GPU”的 AI 药物发现方案，支持从头分子生成和药物重定位。这反映了 **工具化** 趋势——将计算密集型任务压缩到消费级硬件，降低使用门槛。

5.  **科学绘图 Agent 工具兴起**：`AgentFigureGallery` 为 Claude Code、Codex 等编码 Agent 提供“即插即用”的科学绘图技能，通过视觉参考和人类反馈指导代码生成。这是 **LLM Agent 化** 在科研辅助工具中的新应用，旨在解决 Agent 生成图表质量不可控的问题。

**趋势判断**：本期高星区呈现 **“底层 Rust 化 + 上层 Agent 化”** 的双轮驱动格局。生物信息学工具正经历从解释型语言到编译型语言的性能革命，同时 LLM Agent 正从“聊天机器人”进化为“领域专家助手”，在变异解读、细胞注释等具体任务上提供可审计的推理能力。基因组基础模型向作物领域的拓展预示着 AI 育种时代的加速到来。
